# Jogo da Adivinhação

Bem-vindo ao projeto **jogo-da-adivinhacao**! Este é o meu primeiro commit, e embora o programa esteja em desenvolvimento, ainda não é possível jogar. No entanto, em breve, você poderá desfrutar deste emocionante jogo.

O projeto consiste em um jogo de adivinhação, onde você tenta adivinhar um número secreto ou um país, dependendo da versão em que ele está sendo desenvolvido. O jogo incorpora dois módulos Python adicionais: `numeros.py` e `paises.py`.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

- **app_adivinhacao.py**: Este é o programa principal do jogo da adivinhação. Ele importa os outros dois programas (`numeros.py` e `paises.py`) para adicionar funcionalidades ao jogo.

- **numeros.py**: Este módulo contém funções relacionadas à geração de números aleatórios, que são usados para definir o número secreto no jogo.

- **paises.py**: Este módulo inclui uma lista de países que pode ser usada para criar uma lista de opções para que o jogador escolha o país que ele está tentando adivinhar.

## Como Jogar

Embora o jogo ainda não esteja funcional, em breve você poderá executar o programa `app_adivinhacao.py` no seu ambiente Python para começar a jogar. O jogo lhe pedirá para adivinhar um número ou um país, dependendo da versão em que ele está sendo desenvolvido. Siga as instruções na tela e tente acertar o número ou país secreto.
